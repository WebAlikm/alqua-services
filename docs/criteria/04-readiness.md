# Criterion 4: Readiness of the Solution

## Scoring Target

Target band: **8-10 / 10**

The repo should demonstrate a working end-to-end resident flow, not just a concept.

## Evidence Requirements

- Working prototype.
- Clear run instructions.
- Demo video or screenshots.
- Sample data that proves the flow.
- No broken core path.

## Project-Specific Examples

End-to-end flow:

1. Resident opens AlQua'a Services.
2. Resident selects an audience or category.
3. Resident sees matching services, events, or opportunities.
4. Resident opens contact, WhatsApp, or map action.

## Checklist

- [ ] App loads locally.
- [ ] Search works.
- [ ] Filters work.
- [ ] Listing cards show useful details.
- [ ] Contact/map actions exist.
- [ ] Demo video proves the core flow.
