# Criterion 2: Relevance to the Challenge

## Scoring Target

Target band: **9-10 / 10**

The project should be unmistakably aligned with **Challenge 4: Connecting residents to services, opportunities and events**.

## Evidence Requirements

- State the chosen challenge directly.
- Connect the feature set to the challenge language.
- Avoid drifting into a generic directory or unrelated civic app.

- Filters match resident needs: family, youth, entrepreneurs, farm owners, visitors, and older residents.
- Listings include contact and map actions so discovery can become action.
- Arabic-first UX supports the local resident context.
