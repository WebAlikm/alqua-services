# Deployment

## Initial Plan

The first prototype will be deployable as a static web app.

Preferred options:

- GitHub Pages for simple public hosting from the repository.
- Vercel for preview deployments and quick iteration.

## Cost Assumption

The hackathon prototype can run on free static hosting. A future production version may add a managed database, authentication, and notification provider.

## Verification

Deployment verification should include:

- Public URL loads on mobile and desktop.
- Search and filters work with sample data.
- Listing contact actions open the expected channel.
- README contains the live URL and demo video link.
